BEM 命名規則

2017-06-30 未分類

BEMは「Block」「Element」「Modifier」の頭文字をとったもの

Block(ブロック)

  • 構成の「元」となる要素。
  • 必ずトップに来る。

Element(要素)

  • Blockに含まれる要素。Blockの子。
  • 「__」（アンダースコア2つ）で区切る。

Modifier(状態)

元となるものから状態が変化した物表す。 * 「–」（ハイフン2つ）で区切る。

例

.content__text

「.content」に含まれている、「.content__text」

<div class=".content">
    <div class=".content__text"></div>
</div>

.content__text–red

「.content__text」の赤色バージョン

.content__text : {}
.content__text--red :{color:read;}
