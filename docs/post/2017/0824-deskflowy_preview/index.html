とりあえす、Mac版のみですが、DeskFlowyを公開します。

DeskFlowyとは

DeskFlowy は WorkFlowy 専用のクライントソフトです。 WorkFlowy 本家にはない機能をたくさん追加してあり、とても便利につかえるとおもいます。

僕は、 DeskFlowy を普段使いしていて、特に問題なく使用しています。おかげでブラウザでWorkFlowyにアクセスすることはほとんど無いぐらいに使っています。

公開方法をどうしようか、、AppStorから出そうかどうしようかと悩んでましたのですが、ストアを使うと、色々と費用や手間がかさむし、 長々と悩んでいてもしかたないので、でとりあえず、カンパウェアとして公開することにしました。

皆さんの WorkFlowy ライフがさらに快適になることを願ってます。

DeskFlowyのダウンロード

DeskFlowy のサイトから、DLして、インストールして下さい。

DeskFlowy

インストール方法（Mac）

  1. DLしたファイルを開きます。
  2. DeskFlowy のアイコン を、「アプリケーション」フォルダにドラッグ＆ドロップ。

Windows版のDeskFlowyについて

DeskFlowy はWindowsでも動かすことが可能です。実際に数人の方のところで、動いているようです。

しかし、 不具合も多いと思います。

というのも、まともに動くWindowsPC1を持っていないために、僕のところで動作確認ができないからです。 現段階では、特にバージョンアップ（自動アップデートも動かないし）時に、不具合が起きるようです。 どなたかがポンと、WindowsPCを提供していただけたら、Windows版 DeskFlowy を仕上げるモチベーションもあがって、早く公開できるようになるかと思います。(^_^;)

カンパをお願いします

DeskFlowy のバージョンアップやサポートの為には、開発ツール（ハードやソフト）のアップデートや維持、使用料などの費用がかかっています。

今後のバージョンアップや、皆様からのご要望の追加やその開発の費用に充てるためにも、暖かいご支援をよろしくお願いいたします。

いただきましたご支援は、今後の開発のために役立てさせていただきます。& 私の開発モチベーションも上がります。

まずは、ほぼ完成しているWindows版を完成する為に必要なWindowPCの費用に充てたいと思ってます。

なお、誰がカンパしたとなどの個人情報などは一切、公表いたしませんし、こちらから連絡したりすることもありません。 また、カンパの有無で機能がわかる等もありません。
